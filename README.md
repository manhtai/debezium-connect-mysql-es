# Prebuilt Kafka Connect image using MySQL Debezium and Elasticsearch connectors.

- Debezium 0.9
- MySQL 5.x
- Elasticsearch 2.x, 5.x, 6.x

## Usage:

```
docker pull manhtai/debezium-connect-mysql-es:0.9
```
